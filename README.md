# Big Day To-Do 📝

Um aplicativo de lista de tarefas moderno, elegante e funcional desenvolvido com Flutter. O *app utiliza conceitos de *Glassmorphism* para uma interface visualmente atraente, proporcionando uma experiência de usuário fluida e intuitiva.

## ✨ Funcionalidades

- **Gestão de Tarefas:** Crie, edite, marque como concluído e remova tarefas com facilidade.
- **Categorização:** Organize suas tarefas em categorias como Trabalho, Pessoal, Saúde e Estudo.
- **Dashboard de Produtividade:** Visualize seu progresso geral e estatísticas por categoria em tempo real.
- **Interface Glassmorphism:** Design moderno com efeitos de transparência e desfoque.
- **Busca em Tempo Real:** Encontre rapidamente qualquer tarefa pelo título.
- **Feedback Háptico:** Respostas táteis integradas para ações importantes, aumentando a imersão.
- **Persistência de Dados:** Suas tarefas são salvas localmente utilizando `shared_preferences`.
- **Animações Fluidas:** Transições e interações suaves em toda a interface.

## 🚀 Tecnologias Utilizadas

- **Flutter:** Framework principal para desenvolvimento cross-platform.
- **Dart:** Linguagem de programação robusta e eficiente.
- **Shared Preferences:** Armazenamento local persistente.
- **Haptic Feedback:** Integração com hardware para respostas táteis.

## 🛠️ Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/bergson-dev/lista-de-notas.git
    ```
2.  **Navegue até o diretório do projeto:**
    ```bash
    cd lista-de-notas
    ```
3.  **Instale as dependências:**
    ```bash
    flutter pub get
    ```
4.  **Execute o aplicativo:**
    ```bash
    flutter run
    ```

## 📂 Estrutura do Projeto

O código principal reside em `lib/main.dart`, onde estão definidos:
- O modelo de dados `Task`.
- A lógica de persistência e navegação.
- As telas de Lista de Tarefas e Dashboard.
- Componentes de UI customizados.
